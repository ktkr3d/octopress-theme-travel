# octopress theme travel

my octopress theme.

## Install

```
$ cd /path/to/your/octopress
$ git clone http://github.com/ktkr3d/octopress-theme-travel.git .themes/octopress-theme-travel
$ rake install['octopress-theme-travel']
$ rake generate
```
